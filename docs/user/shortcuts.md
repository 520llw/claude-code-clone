## Shortcut Overview

Keyboard shortcuts allow you to work more efficiently in Claude Code Clone. Shortcuts are organized by category and can be customized to match your preferences.

### Shortcut Notation

- `Ctrl` - Control key
- `Alt` - Alt/Option key
- `Shift` - Shift key
- `Cmd` - Command key (macOS)
- `↑` `↓` `←` `→` - Arrow keys
- `Enter` - Return/Enter key
- `Esc` - Escape key
- `Tab` - Tab key
- `Space` - Spacebar

### Viewing Shortcuts

```
> /help shortcuts
> Show keyboard shortcuts
```

## Custom Shortcuts

### Creating Custom Shortcuts

Add custom shortcuts in your configuration:

```json
{
  "shortcuts": {
    "custom": {
      "Ctrl+Shift+T": {
        "command": "/test",
        "description": "Run tests"
      },
      "Ctrl+Shift+B": {
        "command": "/shell 'npm run build'",
        "description": "Build project"
      },
      "Ctrl+Shift+L": {
        "command": "/lint --fix",
        "description": "Lint and fix"
      }
    }
  }
}
```

### Shortcut Configuration File

```json
{
  "shortcuts": {
    "enabled": true,
    "vim_mode": false,
    "emacs_mode": false,
    "custom": {
      "keybinding": {
        "command": "action",
        "description": "Description"
      }
    },
    "disabled": [
      "Ctrl+Q"
    ]
  }
}
```

### Vim Mode

Enable Vim keybindings:

```json
{
  "shortcuts": {
    "vim_mode": true
  }
}
```
